Komunitas Ubuntu Indonesia

Komunitas Ubuntu Indonesia adalah Ubuntu LoCo (Local Community) di Indonesia. Ini adalah tempat untuk semua orang yang tertarik menggunakan, mengembangkan, atau berkontribusi pada proyek Ubuntu.

Ubuntu 22.04 LTS

Ubuntu 22.04 LTS Jammy Jellyfish is the latest version released in Aptril 2022. You can get it today, and run it on various environments such as Desktop, Servers, IoT, Cloud and much more.
